## Project Description
Data mart ini dirancang untuk Fakultas Sains guna memonitor kinerja akademik, penelitian, dan publikasi dosen.

## Business Domain
Fakultas Sains berfokus pada kegiatan tridharma perguruan tinggi: pendidikan, penelitian, dan pengabdian masyarakat.

## Architecture
- Approach: Kimball
- Platform: SQL Server on Azure VM
- ETL: SSIS

## Key Features
- Fact Tables: AcademicPerformance, ResearchOutput
- Dimension Tables: Student, Course, Instructor, Program, Date
- KPIs: IPK rata-rata, publikasi dosen, rasio dosen:mahasiswa
